title: 'Le "Deployment as a service" (DaaS) ou comment transformer son "infrastructure as code" en véritable API et produit'
speakers:
  - idriss-neumann
language: french
---

On va voir ensemble comment on peut utiliser Pulumi, une technologie concurrente de Terraform qui permet de faire de l'infrastructure as code avec son 
langage préféré. Et on verra comment cette différence nous a permis de construire une véritable API de déploiement agnostique des différents fournisseurs 
de cloud.
