Vous entendez parler d'OpenTelemetry (OTel) à toutes les sauces, mais vous voulez enfin voir ce qu'il a dans le ventre ? Rejoignez-nous pour un atelier 100 % hands-on, conçu pour vous donner les clés de l'observabilité moderne sur Kubernetes. On va manipuler, configurer et... casser des choses (parce que oui, shit happens !).

Au programme de ce workshop :

Déploiement initial : Comment installer proprement OpenTelemetry dans un cluster Kubernetes.

Le mode "Pompier" : Debugger son architecture OTel quand les métriques et les traces ne remontent pas (spoiler : ça arrive même aux meilleurs).

La magie de l'auto-instrumentation : Collecter des signaux sans toucher à une seule ligne de code.

Le sur-mesure : Ajouter de l'instrumentation manuelle pour capturer la logique métier qui compte vraiment pour vous.

Prérequis :

Cet atelier est un vrai hands-on. Pas de spectateurs, que des acteurs !

Pensez absolument à venir avec votre laptop.

Aucun stress pour la data : une connexion Internet (Wi-Fi) vous sera fournie sur place.

Prêts à dompter vos flux de données ? On vous attend ! 🚀
